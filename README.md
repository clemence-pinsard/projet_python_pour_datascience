# Projet python pour la datascience

## Nos données

Inégalités sociales face aux maladies chroniques (ER 1243) 

https://data.drees.solidarites-sante.gouv.fr/api/explore/v2.1/catalog/datasets/er_inegalites_maladies_chroniques/exports/csv?use_labels=true

## Structure de notre dépôt

### Fichier principal

Notre ficher principal est le Jupyter Notebook intitulé main.ipynb.

### Fichier d'installation des packages nécessaires

Le fichier requirements.txt contient tous les packages nécessaires au projet et qu'il faudra installer (la cellule pour les installer est présente dans le notebook).

### Fichier avec les fonctions annexes

Le fihcier fonctions.py contient toutes les fonctions annexes nécessaires au projet qui sont importées dans le notebook.

## Lancer notre projet 

Ouvrir le notebook projet.ipynb et cliquer sur Run All.
